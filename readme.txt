Pascal’s Triangle :
You are given a number n, you are required to print the first n rows of the Pascal triangle.
 
Your task is to complete the function Pascal triangle which receives n as the parameters and returns a 2D array containing the first n rows of Pascal’s triangle.
Input Format
You are given a single integer n representing the number of rows of Pascal’s triangle.
Output Format
Print n lines of output, where the ith line represents the ith row of the triangle.
Example 1
Input
3
Output
1
1 1
1 2 1
Example 2
Input
5
Output
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
Constraints:
1<= n <= 30
